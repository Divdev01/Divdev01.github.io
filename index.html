Using Image Analysis to Estimate the Density of Blood Cells

Using AI and NLP to Automate Dubbing Process for Improved Accessibility and Inclusivity

Analysis on Facebook's rebranding to meta using Tweets

  • Preprocessing: removed punctuations, tags, @usernames,numbers, emojis, stopwords
  • Embeddings: Word2vec, GloVe, Doc2vec, BERT
  • Clustering: K-means Clustering, Agglomeratve clustering
  • BERT along with agglomerative clustering gave good interpreting results with no overlap among clusters
  • Overall people were talking more about the hiring because of the metaverse and the virtual reallity and with few negative opinions

Developing an Engine for Heart Failure Detection using Deep Learning

  • Reviewed research papers and incorporated few methodologies from relevant papers.
  • Worked on preprocessing to remove the noise in the signal and performed normalization
  • RandomOverSampler was used to combat class imbalance
  • Designed a neural network with 1D CNN blocks based on the research papers
  • Deployed the selected model in a streamlit app and delivered to the client

A text classifier to identify violence in online chats with children

  • Preprocessing: removed urls, hashtags and extra space.
  • Model : Utilized CamemBERT from the Transformers library and fine-tuned it to our dataset
  • Performance: Achieved 86% accuracy, 84% precision, and 84% recall
  • Deployment: Deployed a proof of concept (POC) in a Streamlit app.

Analysis on types of dwellings, population and consumer price index in Canada

  • Housing data, population and immigration data were collected from statistics canada website.
  • Interative plots are created using Plotly to show province wise visualization of dwelling, population and consumer price index.
  • Home owners home and mortgage insurance is the highly inflated item which is followed by tobacco products

Proactive Financial Planning for Lung Cancer Patients

  • Built a predictive model to estimate the treatment cost for Lung Cancer patients and deployed a proof of concept in a Streamlit application.
  • Extensive cleaning was done to fix the discrepancies in certain fields and EDA was done to find insights from the data.
  • Features were selected based on correlation, relevance to target and user related relevance.
  • Outliers were removed to reduce the noise
  • Then target guided ordinal encoding and One hot ecoding was performed and Xboost algorithm was used to build a predictive model.
  • Finally, hyper parameters were tuned with bayesian optimisation using optuna library.

Chatbot-Smart guide for Canadian immigration applicants

Analysis on housing supply and demand, construction and population in Canada
